A website with multiple mini-games w/ rooms (pun intended).

- Server: NodeJS + [Fastify](https://www.fastify.io/)
- Client: Vue3

## How to start (dev mode)

1. Install the dependencies:

```sh
cd ./backend/
npm install

cd ../frontend/
npm install
```

2. Run the backend:

```sh
cd ./backend/
npm run dev
```

3. Run the frontend:

```sh
cd ./frontend/
npm run dev
```
